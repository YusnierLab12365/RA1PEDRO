Automatización de despliegue - RA1

Profe, no podemos realizar el RA con la página web de nuestro proyecto debido a que contiene PHP para el funcionamiento de algunas animaciones, conexión con la base de datos, sistema de login, consultas dinámicas, entre otras funcionalidades necesarias para su correcto funcionamiento.

Debido a estas limitaciones con GitHub Pages, realizamos esta pequeña página web con puro HTML y CSS, demostrando el proceso de despliegue, automatización de pruebas y uso de GitHub Actions sin incluir PHP.
